# Lab Simulator

A Python-based cybersecurity simulation that generates system logs, simulates brute-force attacks, and detects suspicious activity using rule-based logic.

## Features
- Log generation system (login success/fail, file access)
- Attack simulator (brute force login attempts)
- Threat detection engine
- Severity-based alert system (LOW / MEDIUM / HIGH)
- Log file storage for analysis

## Stack
- Python

## How to Run
bash
python main.py
